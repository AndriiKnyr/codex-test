# TOTP API Example

This repository contains a .NET 9 Web API that validates TOTP codes using the `Otp.NET` library. The solution demonstrates usage of controllers, MediatR with a CQRS approach, dependency injection, and Active Directory integration.

## Projects
- **TotpApi** – Web API application.
- **TotpApi.Tests** – xUnit tests.

## Running Tests

The environment used for this example might not include the .NET SDK. If `dotnet` commands are unavailable, tests cannot be executed.
